# Task Tracker CLI Python

Aplikasi command line interface (CLI) sederhana yang dibangun dengan Python untuk melacak, memperbarui, dan mengelola tugas. Data disimpan dalam file JSON.

## Cara Menggunakan

1. **Instalasi:** Pastikan Python sudah terinstal.
2. **Jalankan:** Gunakan `python task_cli.py <aksi> [argumen]`.

## Daftar Perintah

* `add "deskripsi"`: Menambah tugas baru.
* `list [status]`: Mencantumkan semua tugas atau berdasarkan status (todo, inProgress, done).
* `update <id> "deskripsi baru"`: Memperbarui deskripsi tugas.
* `mark-done <id>`: Menandai tugas sebagai selesai.
